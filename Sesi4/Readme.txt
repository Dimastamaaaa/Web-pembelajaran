// algoritma untuk menghitung luas dan keliling lingkaran menggunakan JavaScript:

1.Tentukan jari-jari lingkaran (r)
2.Hitung luas lingkaran (A) dengan menggunakan rumus A = πr^2.
3.Hitung keliling lingkaran (C) dengan menggunakan rumus C = 2πr.
4.Tampilkan hasil luas dan keliling lingkaran.

// penjelasan

Pada kode di atas, kita mendefinisikan variabel r sebagai jari-jari lingkaran,
 kemudian menghitung luas dan keliling lingkaran dengan menggunakan rumus yang sudah dijelaskan sebelumnya.
  Kita juga menggunakan toFixed() untuk membulatkan hasil menjadi 2 angka di belakang koma
   dan menampilkan hasil dengan menggunakan console.log().
